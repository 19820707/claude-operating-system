# Project bootstrap examples

| Directory | Purpose |
|-----------|---------|
| **`minimal/`** | Core `.claude/` layout, session memory stubs, command/policy/skill placeholders, adapter map, sample decision-log JSONL. |
| **`advanced/`** | Adds stack-profile and invariant-engine placeholders plus session-memory overview. |

These trees **illustrate** `init-project.ps1` output; they are not produced by running init into this path. For the canonical path list, see `bootstrap-manifest.json` in the OS repo root.
