# Minimal bootstrap example (reference)

This folder is **not** a runnable project. It shows the **shape** of a repository after `init-project.ps1` from Claude OS: which directories exist, where session memory lives, how decision log lines look, and where adapters land.

- Canonical bootstrap behavior: `init-project.ps1` in the OS repo root.
- Full critical path list: `bootstrap-manifest.json` → `projectBootstrap.criticalPaths`.
- Operational guide: `docs/PROJECT-BOOTSTRAP.md`.

All content here is **synthetic** (no secrets, no machine-specific paths, no real tokens).
